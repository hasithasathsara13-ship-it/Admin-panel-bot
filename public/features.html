Velo.ai Platform

Complete WhatsApp Business automation with AI-powered chat, bulk messaging, product management, and multi-tenant admin panel.

💬

WhatsApp Business Chat

Core

Real-time Messaging

Send and receive WhatsApp messages in real-time with Supabase Realtime subscriptions. Messages appear instantly without page refresh.

Core

24-Hour Window Management

Automatic detection of WhatsApp's 24-hour messaging window. Input disables when window closes, prompts to use template messages.

Core

Template Messages

Send pre-approved Meta template messages to re-engage customers after the 24-hour window expires. Supports hello_world and custom templates.

New

Dynamic Meta Templates

Fetches live templates from Meta Cloud API. Shows template name, language, and approval status. Auto-populates dropdown for sending.

Core

Image Sending & Receiving

Send images to customers and view received images with hover-to-download functionality. Supports JPEG, PNG, and WebP.

New

Send Image by URL

Send product images or review screenshots to customers via direct URL without needing to re-upload. Used for reviews and catalog sharing.

Core

Voice Notes

Record and send voice messages directly from the dashboard. Receive and play back customer voice notes with audio player.

Core

Document Support (PDF, Word, Excel)

View and download documents sent by customers. Shows document type icon, file label, and one-click download button.

Core

Video Support

Receive and play video messages from customers with built-in video player in chat bubbles.

Core

Message Reply & Edit

Reply to specific messages with quoted preview. Edit sent messages directly from the dashboard with edit history tracking.

Core

Unread Message Indicators

WhatsApp-style unread badges with count. Bold text and green timestamp for conversations with unread messages.

Core

Contact Management

Save customer phone numbers with custom names. Search contacts by name or number in the conversation list.

Core

Bot Toggle (Human Handoff)

Toggle AI bot on/off per conversation. Auto-re-enables after 20 minutes. Visual indicator shows bot status in chat header.

Core

Resizable Panel

Drag-to-resize the conversation list panel on desktop. Adjusts between 15% and 50% width for optimal workflow.

New

Send Reviews via Chat

One-click send all review screenshots and external review link to any customer directly from the + menu in the chat interface.

New

Long-Press Context Menu

Touch-friendly long-press or right-click context menu on messages for reply, edit, and delete actions. Works on mobile and desktop.

Template Management

New

View All Templates

Dedicated page listing all Meta-approved message templates with status badges (Approved, Pending, Rejected), category, and language.

New

Create Templates

Create new message templates with header (text/image/video/document), body text with variables, footer, and CTA buttons directly from the dashboard.

Delete Templates

Template Preview

Bulk Broadcast

Feature

Audience Filtering

Filter contacts by activity: All contacts, active in 7 days, active in 30 days, inactive 30+ days for re-engagement campaigns.

Feature

Manual Number Entry

Paste phone numbers directly or type them in. Supports comma-separated, semicolon-separated, or one per line format.

Feature

Spreadsheet Import (.csv / .xlsx)

Upload CSV or Excel files to extract phone numbers automatically. Numbers are parsed from the first column.

Feature

Template Broadcast

Send pre-approved template messages to multiple contacts at once. Now fetches live templates from Meta API with approval status.

Feature

Custom Text Broadcast

Send freeform text messages to recently active contacts (within 24h window). Up to 4096 characters.

Feature

Select/Deselect Recipients

Cherry-pick specific contacts or select all. Search and filter within the contact list before sending.

Feature

Rate-Limited Sending

100ms delay between messages, max 500 recipients per broadcast. Prevents Meta API throttling and account restrictions.

Feature

Broadcast Results & Error Reporting

Detailed results showing sent/failed counts with per-number error details. Broadcast history logged to database.

Safety

Ban Risk Warning

In-app warning modal when using manual entry. Explains account ban risks and WhatsApp policy requirements before proceeding.

Products & Reviews

Feature

Product Catalog

Add, edit, and delete products with name, description, price, category, stock count, and up to 4 images per product.

Feature

Image Upload with Drag & Drop

Upload product images via drag-and-drop or file picker. Images stored in Supabase Storage with public URLs.

Feature

Product Filters

Filter by All Products, Featured, On Sale, or Out of Stock. Search products by name or category.

Feature

Stock Management

Track stock counts with low stock alerts (below 5 units). Support for unlimited stock option.

Feature

Customer Reviews (Screenshots)

Upload up to 6 review screenshots. Add optional external reviews link (Google, Facebook). Send reviews to customers via WhatsApp.

Feature

Send Reviews via WhatsApp

One-click send all review screenshots and review link to any customer directly from the chat interface.

Feature

Complete Storage Cleanup

Deleting product images or review screenshots removes both the database reference and the actual file from Supabase Storage.

Feature

Customer Insights

Analytics Dashboard

💳

Billing & Subscriptions

Billing

Plan Tiers (Starter, Growth, Scale)

Three pricing tiers with monthly/yearly billing cycles. Message quotas and feature limits per plan.

Billing

Grace Period & Hard Block

Configurable grace days for past-due accounts. Hard block messaging when quota exceeded. Courtesy WhatsApp to owner at 100% usage.

Billing

Usage Tracking

Automatic message counting per billing period. Increment on each inbound WhatsApp message via database RPC.

Billing

Mark Paid

Admin can mark subscriptions as paid, resetting billing cycle and re-enabling messaging for the workspace.

Feature

Supabase Auth (Email/Password)

Secure login with Supabase Auth. Auto-creates business workspace on first login. Session persistence with token refresh.

Feature

Multi-Tenant Architecture

Each business has isolated data (products, orders, messages, customers). RLS policies enforce data separation at the database level.

Feature

Desktop Notifications

Browser push notifications for new orders and human handoff requests. Toggle on/off from the dashboard.

Feature

Dark/Light Theme

Admin theme toggle stored in localStorage. Full dark mode support across all dashboard components.

Feature

Responsive Design

Full mobile support with bottom navigation bar, collapsible sidebar on desktop, and touch-optimized interactions.

Feature

WhatsApp Webhook Integration

Receives inbound messages, media, documents, voice notes, locations, and interactive replies. Signature verification with META_APP_SECRET.
